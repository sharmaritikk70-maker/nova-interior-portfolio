NOVA Interior Website - Update Guide

Folder structure:
- index.html = page content and form options
- style.css = design, colors, layout, Apple-style smooth animations
- script.js = image slider, form steps, WhatsApp submit
- assets/hero/ = all slider render images
- assets/pdf/ = project requirement PDF, if needed

How to update images in future:
1. Replace images inside assets/hero/.
2. Keep same names like 01-render.jpg, 02-render.jpg etc.
3. 01-render.jpg appears first in slider.

How to change WhatsApp number:
Open script.js and replace:
917057047957
with your new number including country code.

How to change Instagram:
Open index.html and replace:
https://www.instagram.com/novainteriordesign70/

Important:
WhatsApp auto-send is not allowed by WhatsApp. Form submit opens WhatsApp with message ready. Client must press Send.
